# SkillCraft Task 04 - Marketing Campaign EDA

## Beyond the Clicks

Every click represents customer engagement, but the real value lies in understanding the story behind the data. This project focuses on analyzing marketing campaign performance to uncover insights that support smarter business decisions.

### Project Objective
Perform Exploratory Data Analysis (EDA) on marketing campaign data to evaluate campaign effectiveness, conversion trends, audience engagement, and ROI.

### Key Insights
- ROI Analysis by Marketing Channel
- Conversion Rate Analysis
- Campaign Type Performance
- Target Audience Analysis
- Location-Based Insights
- Interactive Dashboard Filtering

### Tools Used
- Power BI
- Data Analytics
- Data Visualization
- Business Intelligence

### Business Value
The analysis helps identify high-performing marketing channels and supports data-driven budget allocation and marketing strategy decisions.
